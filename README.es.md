# Centro OSINT de Inteligencia sobre Cáncer de Páncreas

Plataforma autónoma de inteligencia de código abierto (OSINT) para el cáncer de páncreas.
Agrega noticias clínicas, ensayos, literatura y recursos de apoyo a pacientes;
aplica traducción por IA, etiquetado de entidades y puntuación de riesgo; y ofrece un
hub personalizado donde el perfil des-identificado del paciente genera un feed
a medida de noticias, artículos recientes, artículos de PubMed, ensayos en reclutamiento,
búsqueda de pruebas y un asistente AI fundamentado.

## Inicio rápido

```bash
npm install
cp .env.example .env   # luego rellena las claves (normalmente opcionales)
npm run dev            # http://localhost:3000
```
